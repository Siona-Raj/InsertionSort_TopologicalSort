Insertion Sort - Pseudocode


insertionSort(A):
    for i ← 1 to length(A) - 1:
        key ← A[i]
        j ← i - 1

        while j ≥ 0 and A[j] > key:
            A[j + 1] ← A[j]
            j ← j - 1

        A[j + 1] ← key

Explanation

The insertion sort algorithm builds the sorted array one item at a time:

  • Start from the second element, assume the first element is already sorted.
  • Compare the current element (key) with those before it.
  • Shift larger elements to the right to make space for the key.
  • Insert the key in its correct sorted position.
  • Repeat for all elements until the array is sorted.
