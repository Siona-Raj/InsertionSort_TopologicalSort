Depth-First Search (DFS) - Pseudocode


DFS(node, visited):
    if node in visited:
        return

    visit(node)
    visited.add(node)

    for neighbor in adjacentNodes(node):
        DFS(neighbor, visited)

Explanation

Depth-First Search (DFS) is a graph traversal algorithm that explores as far as possible along each branch before backtracking:

  • Start from a selected node.
  • Mark it as visited and perform any required operation (like printing).
  • Recursively visit each unvisited neighbor.
  • This process continues until all reachable nodes are visited.
  • DFS is typically implemented using recursion or a stack.
